
## 前言
简介：首页快捷键，类似支付宝、美团、饿了么首页快捷键，传入数据即可使用，可传背景色，一行展示个数等

## 有疑问
微信搜索“慢慢向好”小程序，找客服反馈，相应问题。
				  

## 素材
[图片资源](https://pixabay.com)
